## 第1集：向量

**观看日期：2026.6.2**

向量有三种视角，线性代数里用的是坐标视角，但你要始终带着几何箭头的直觉。

- 物理视角：有长度有方向的箭头
- 计算机视角：一串数字（坐标）
- 数学视角：满足加法与数乘运算的对象

向量的两种基本运算：
- 加法：箭头首尾相连，几何上是平移
- 数乘（标量乘法）：拉伸或缩短箭头，负值则反向

**点乘（Dot Product）：**
公式：`a·b = |a||b|cosθ`
- 几何意义一：a 投影到 b 方向上的长度 × b 的长度
- 几何意义二：两个向量方向的相似度（cosθ 的体现）
- 点乘 > 0 → 夹角小于90°，同向趋势
- 点乘 = 0 → 垂直
- 点乘 < 0 → 夹角大于90°，反向趋势

游戏里的应用：判断敌人是否在视野内（面朝方向点乘敌人方向）、计算光照强度（法线点乘光线）。

手电筒照墙，光线方向就是投影方向。

**叉乘（Cross Product）：**
- 输入两个向量，输出一个**垂直于它们所在平面**的新向量
- 方向用右手定则：食指=a，中指=b，拇指方向=结果方向
- 长度 = a和b构成的平行四边形的面积
- 叉乘不满足交换律：a×b = -(b×a)

## 第2集：线性组合、张成空间与基向量

**核心：基向量是空间的标尺，线性组合就是「用几份i + 几份j」定位空间中任意一点。**

- 基向量 `i=(1,0)` 和 `j=(0,1)` 是坐标系的标尺
- 任意向量 `(x,y)` 其实就是 `x·i + y·j`
- 线性组合：拿几个向量分别缩放再相加，得到的结果
- 张成空间（span）：一个或一组向量通过所有可能的线性组合，能到达的全部位置
  - 一个向量的 span = 一条线
  - 两个不共线的向量的 span = 整个平面
  - 两个共线的向量 = 还是一条线（冗余）
- 线性相关：一组向量中，至少有一个可以被其余的组合出来（有多余的）
- 线性无关：每个向量都为张成空间贡献了新的维度

## 第3集：线性变换与矩阵

**核心：矩阵不是一堆数字，而是一种空间变换——把空间「拉扯/旋转」到新位置。矩阵的列就是变换后的基向量。**

- 线性变换 = 满足两条规则的函数：(1)直线保持直线 (2)原点不动
- 线性变换分三类：旋转、缩放、剪切
- **记一个变换不需要记所有点去哪了，只需要记基向量 `i` 和 `j` 变换后去哪了**
- 矩阵 `[[a, b], [c, d]]` 中：
  - 第一列 `(a,c)` = 变换后的 `i`
  - 第二列 `(b,d)` = 变换后的 `j`
- 矩阵×向量 = 用变换后的基向量重新组合目标点：`x·(新i) + y·(新j)`
- 2D旋转矩阵：`[[cosθ, -sinθ], [sinθ, cosθ]]`

## 第4集：矩阵乘法与复合变换

**核心：两个矩阵相乘 = 先后做两次变换，从右往左读。**

- `AB` 表示：先应用 B 变换，再应用 A 变换（从右往左读）
- 这就是复合变换——把两个变换「压缩」成一个矩阵
- 矩阵乘法不满足交换律：AB ≠ BA（先旋转再剪切 ≠ 先剪切再旋转）
- 矩阵乘法满足结合律：(AB)C = A(BC)
- 几何理解记忆法：A的列是「变换后的基向量」，A×B 就是「把B的每一列当成向量，分别做A变换」

---

## 第5集：三维空间中的线性变换

**核心：3D和2D逻辑完全一样，矩阵的列就是变换后的基向量 `i, j, k`。**

- 3D空间基向量：`i=(1,0,0)`, `j=(0,1,0)`, `k=(0,0,1)`
- 3×3矩阵的三个列 = 变换后的 `i, j, k`
- 绕X轴旋转：X列不变(1,0,0)，Y和Z列在各自的平面里旋转
- 绕Y轴旋转：Y列不变(0,1,0)，X和Z列旋转
- 绕Z轴旋转：Z列不变(0,0,1)，X和Y列旋转
- 三个旋转矩阵不一样，因为**旋转轴对应的那个基向量不动，另外两个在动**

---

## 第6集：行列式

**核心：行列式 = 线性变换对面积的缩放倍率。正值保方向，负值翻转空间。**

- 行列式的几何意义：变换后单位正方形的面积（2D）或体积（3D）
- det > 0：空间方向不变（面积缩放）
- det = 0：空间被压缩成线或点（降维了），矩阵不可逆
- det < 0：空间被翻转（镜像），绝对值为面积缩放倍率
- 3D行列式同理，是单位立方体变换后的体积
- 行列式=0意味着变换把空间压缩到了更低维度，信息丢失，无法还原

---

## 第7集：逆矩阵、列空间、秩、零空间

**核心：逆矩阵 = 把变换「撤销」回去。只有行列式≠0（没降维）的变换才能撤销。**

- 逆矩阵 A⁻¹：A 变换的结果再应用 A⁻¹，回到原始状态
- 列空间：矩阵所有可能的输出向量构成的集合（变换后基向量张成的空间）
- 秩（rank）：列空间的维度数。满秩=没有降维，不满秩=空间被压扁了
- 零空间（kernel）：被变换压到原点的所有向量的集合
- A 可逆 ⇔ det ≠ 0 ⇔ 满秩 ⇔ 没有信息丢失
- 解线性方程组 Ax=b 的几何意义：找一个向量 x，经过 A 变换后落在 b 的位置。如果 A 可逆，x = A⁻¹b；如果 A 不可逆（det=0），要么无解要么无穷多解

---

## 第8集：非方阵

**核心：非方阵 = 在不同维度空间之间做变换。**

- m×n 矩阵：输入 n 维向量，输出 m 维向量
- n 列 = 输入空间有 n 个基向量
- m 行 = 输出空间是 m 维的
- 例如 3×2 矩阵：把 2D 平面映射到 3D 空间中的一张平面（像把纸放在 3D 空间里）
- 例如 2×3 矩阵：把 3D 空间拍扁到 2D 平面上（降维）
