## ① 变换（Transformation）

### L1：课程概览

**核心：图形学不是画图，是模拟光。**

- 四大方向：光栅化、几何、光线追踪、动画模拟
- 为什么学图形学不等于学 OpenGL/Unity/UE5（引擎是工具，原理是底子）
- TA 看重的是「知道 GPU 里发生了什么」

### L2：线性代数回顾

**核心：和 A1 完全重叠，快速过。**

### L3：变换（Model & View）

**核心：3D 物体 → 屏幕，要经历四次变换。**

```
模型空间 → 世界空间 → 视角空间 → 裁剪空间 → 屏幕空间
(Model)    (World)     (View)      (Clip)      (Screen)
    ↓          ↓          ↓
  缩放      摆放位置    LookAt 摆相机
  旋转      组合场景
```

MVP 矩阵 = Projection × View × Model，从右往左读。

- Model 矩阵：把物体从自身坐标搬到世界坐标（旋转+缩放+平移）
- View 矩阵：把整个世界搬到相机面前（就是 LookAt 的逆）
- 两者都是 A1 学过的 4×4 齐次坐标变换

### L4：投影（Projection）

**核心：把3D压到2D，但保留深度信息（给 Z-buffer 用）。**

- 正交投影：直接扔掉 Z，拍平（CAD/策略游戏用）
- 透视投影：近大远小，视锥体挤压成长方体再正交投影
- 两种投影的矩阵写法，本质都是 4×4 齐次坐标

## ② 光栅化（Rasterization）

### L5：三角形光栅化

**核心：屏幕上的每个像素，判断它在不在三角形里面。**

- 为什么用三角形：最简多边形、保证是平面、容易插值
- 光栅化的本质：对每个三角形，遍历它覆盖的像素，填色
- 判断点在三角形内：叉乘（A1 的叉乘直接用上了）
- 采样：屏幕像素是离散的采样点，走样（锯齿）由此而来

### L6：反走样与深度测试

**核心：锯齿是采样不足的问题，深度测试决定谁挡谁。**

- 反走样（Anti-aliasing）：MSAA（多点采样再平均）、FXAA/TAA（后处理模糊）
- 深度缓冲（Z-buffer）：每个像素存最近物体的深度，新的比它近才画
- Z-buffer 是 GPU 硬件支持的，几乎零开销

## ③ 着色（Shading）

### L7：Blinn-Phong 着色模型

**核心：着色 = 算每个像素的颜色。Blinn-Phong 是最经典的入门模型。**

```
颜色 = 环境光 + 漫反射 + 高光
       Ambient + Diffuse + Specular
```

- 环境光：不管光照方向，全场景均匀亮度（最粗暴的近似）
- 漫反射：光打到粗糙表面，均匀散射（Lambert 定律，点乘算夹角）
- 高光：镜面反射方向附近的亮斑（Blinn-Phong 用半程向量简化计算）
- 三种光全是点乘/叉乘在干活

### L8：图形管线与 Shader

**核心：GPU 的流水线长什么样，Shader 在哪执行。**

```
顶点数据 → 顶点着色器 → 图元装配 → 光栅化 → 片元着色器 → 输出
          (Vertex)                (三角形)  (像素)   (Fragment)
```

- Vertex Shader：处理每个顶点（位置变换、传数据给后面）
- Fragment Shader：处理每个像素（算颜色、采样纹理）
- 现代引擎（UE5）里写 Shader = 写这两个阶段的代码
- Shader 用 HLSL/GLSL 写，语法像 C

### L9：纹理映射

**核心：纹理就是给三角形贴壁纸。用 UV 坐标查纹理图上的颜色。**

- UV 坐标：三角形每个顶点存一个 (u,v)，表示纹理图上的位置
- 纹理采样：片元着色器根据插值后的 UV 去纹理图上取颜色
- 问题一：纹理太小 → 双线性插值/Bicubic 平滑放大
- 问题二：纹理太大 → Mipmap（预生成缩小的纹理链，远处用小的）
- 法线贴图/Bump 贴图：用纹理扰动法线，不增加面数也能伪造凹凸

## ④ 几何（Geometry）

### L10-L12：曲线、曲面、网格

（TA 了解即可，不做硬性要求）

- 贝塞尔曲线：控制点定义曲线，de Casteljau 算法
- 曲面：贝塞尔曲面 = 两个方向的贝塞尔曲线
- 网格操作：细分（加面）、简化（减面）、正则化（把三角面变均匀）
- 阴影映射：从光源视角渲染一张深度图，判断哪些像素在阴影里

## ⑤ 光线追踪（Ray Tracing）

### L13-L17：从 Whitted 到路径追踪

（TA 了解原理即可）

- 光线追踪 vs 光栅化：RT 从眼睛反向追踪光线，光栅化从物体正向投影
- Whitted 风格：反射+折射+阴影，递归追光
- BVH：用包围盒加速光线和三角形的求交（砍掉大量不必要的计算）
- 辐射度量学：精确描述光的物理量（辐射通量、辐射度、亮度）
- BRDF：描述材质如何反射光（入射方向 → 出射方向的分布）
- 蒙特卡洛路径追踪：随机采样光线路径，求积分近似解
- PBR（基于物理的渲染）：真实材质的 BRDF 模型（微表面理论等）
